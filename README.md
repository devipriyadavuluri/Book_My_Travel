# Book My Travel Application

This is a Java-based console application that simulates a travel booking system.

Features:
- View available travel options
- Book tickets
- Cancel bookings
- Display booking details

Concepts Used:
- Core Java
- Object-Oriented Programming (OOP)
- Classes and Packages
- Exception Handling

Technology Used:
- Java

Purpose:
This project was developed to practice Java programming and application design.
